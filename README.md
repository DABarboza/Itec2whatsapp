# Asistente Virtual WhatsApp - ITEC N° 2

Asistente con Inteligencia Artificial para la atención automatizada de la comunidad educativa del ITEC N° 2 Posadas, Misiones.

## 🚀 Tecnologías
- Node.js
- WhatsApp-web.js
- Google Gemini AI (1.5 Flash)

## 🛠️ Instalación
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Crear un archivo `.env` con la `GEMINI_API_KEY`.
4. Ejecutar `node index.js` y escanear el código QR.
